# BANDival Web

Diese App ist das Next.js Frontend + API Layer von BANDival.

## Dev Commands

```bash
npm ci
npx prisma generate
npx prisma db push
npm run dev
```

## Qualitaet

```bash
npm run lint
npm run build
npm run test:e2e
```
